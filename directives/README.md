# Directives

Standard Operating Procedures (SOPs) for the PositivePay Extension project.

Each directive is a Markdown file that describes:
- **Goal**: What the task accomplishes
- **Inputs**: What data or context is required
- **Tools/Scripts**: Which execution scripts to call
- **Outputs**: What is produced
- **Edge Cases**: Known gotchas and error handling

Directives are living documents — update them as you learn.
